# RSI 网格策略设计

## 核心原则
- 永不追高，只在超卖时买入
- RSI 建仓 + 网格跟进
- 严格预算，不超配

## 买入条件

| 条件 | RSI | BB位置 |
|------|-----|--------|
| 建仓 | < 30 | < 30% |

（只有一次建仓，无第二批）

## 卖出条件

| 类型 | 条件 |
|------|------|
| 止损 | -15% |
| 止盈 | +12% |
| 保本 | 浮盈 3%~8% 时回撤到 +0.5% |
| 冲刺 | 浮盈 >8% 时回撤 3% |
| 超买 | RSI >70 或 (RSI>65 且盈利>5%) |
| 超时 | 持仓 >7 天 |

## 资金分配

| 模块 | 金额 |
|------|------|
| RSI 建仓 | $43.75 |
| 网格首单 | $6.25 |
| 网格每层 | $6.25 × 4层 = $25 |
| **合计** | **$75/币** |

说明：
- 网格每笔 = 25 / 4 = $6.25（4层跟进）
- RSI建仓 = 50 - 6.25 = $43.75

## 网格参数

- 层数：8层
- 区间：入场价 ± 8%
- 跟进预算：$25（固定）
- **网格始终开启**

## 监控币种（15个）

BTC, ETH, SOL, DOGE, XRP, ADA, SUI, BNB, LINK, PEPE, AVAX, LTC, TON, ARB, NEAR

## 运行参数

- 扫描间隔：15 分钟
- 持仓检查：1 分钟
- 持仓上限：2 个币种

## 订单流转

- 买单成交 → 更低价挂买单 ("低买")
- 卖单成交 → 更高价挂卖单 ("高卖")
